Convert the following to hex colors

  1. rgb(255, 105, 201)
    • - 255/16 == 15 R15 --> ff
    • - 105/16 == 6 R9 --> 69
    • - 201/16 == 12 R9 --> c9
    • - Answer: #ff69c9
  2. rgb(123, 255, 50)
    • - 123/16 == 7 R11 --> 7B
    • - 255/16 == 15 R15 --> ff
    • - 50/16 == 3 R2 --> 32
    • - Answer: #7bff32
  3. rgb(111, 30, 255)
    • - 111/16 == 6 R15 --> 6f
    • - 30/16 == 1 R14 --> 1e
    • - 255/16 == 15 R15 --> ff
    • - Answer: #6f1eff
  4. rgb(25, 255, 40)
    • - 25/16 == 1 R9 --> 19
    • - 255/16 == 15 R15 --> ff
    • - 40/16 == 2 R8 --> 28
    • - Answer: #19ff28
  5. rgb(10, 255, 55)
    • - 10/16 == 10 --> 0a
    • - 255/16 == 15 R15 --> ff
    • - 55/16 == 3 R7 --> 37
    • - Answer: #0aff37

Convert the following to rgb colors

  1. #ff69c3
    • - ff == 15 * 16 --> 240 + 15 == 255
    • - 69 == 6 * 16 --> 96 + 9 == 105
    • - c3 == 12 * 16 --> 192 + 3 == 195
    • - Answer: rgb(255, 105, 195)
  2. #7bff32
    • - 7b == 7 * 16 --> 112 + 11 == 123
    • - ff == 15 * 16 --> 240 + 15 == 255
    • - 32 == 3 * 16 = 48 + 2 == 50
    • - Answer: rgb(123, 255, 50)
  3. #6f1eff
    • - 6f == 6 * 16 --> 96 + 15 == 111
    • - 1e == 1 * 16 --> 16 + 14 == 30
    • - ff == 15 * 16 --> 240 +15 == 255
    • - Answer: rgb(111, 30, 255)
  4. #19ff28
    • - 19 == 1 * 16 --> 16 + 9 == 25
    • - ff == 15 * 16 --> 240 + 15 == 255
    • - 28 == 2 * 16 --> 32 + 8 == 40
    • - Answer: rgb(25, 255, 40)
  5. #0aff37
    • - 0a == 0 * 16 --> 0 + 10 == 10
    • - ff == 15 * 16 --> 240 + 15 == 255
    • - 37 == 3 * 16 --> 48 + 7 == 55
    • - Answer: rgb(10, 255, 55)
